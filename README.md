## 🚩Project - HYBE

#### 📰 Intro 
HTML, SCSS, Javascript로 개발한 개인프로젝트입니다. GSAP Scrolltrigger를 사용하여 네비게이션 클릭했을 때 해당 섹션으로 넘어가는 UI를 구현하였습니다. Swiper.js에서 destroy()라는 함수를 사용해 특정 해상도에서만 작동되도록 했습니다.
